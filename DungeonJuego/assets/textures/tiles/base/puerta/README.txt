PNGs de puertas.
Tamaño: 64x64 px.

Archivos esperados:
  puerta_h_cerrada.png  -- puerta horizontal cerrada (VAR_PUERTA_H)
  puerta_h_abierta.png  -- puerta horizontal abierta
  puerta_v_cerrada.png  -- puerta vertical cerrada   (VAR_PUERTA_V)
  puerta_v_abierta.png  -- puerta vertical abierta

La variante (H o V) la detecta el editor automaticamente al pintar la puerta.
El programador la lee con mapa->getVariante(fila, col) para elegir que PNG cargar.
